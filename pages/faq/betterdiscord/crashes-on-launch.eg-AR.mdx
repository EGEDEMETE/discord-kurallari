# After installing BetterDiscord, Discord crashes on launch

<Callout emoji="ℹ️">
    This answer will work for Windows users only.
</Callout>

1. Press `Win + R` and type `%APPDATA%/betterdiscord`.
2. Look for these files in the plugins folder and delete them:
```
0PluginLibrary.plugin
0PluginLibrary.plugin.json
0PluginLibrary.config.json
0PluginLibrary.data.json
0PluginLibrary.raw
ZeresPluginLibrary.plugin
ZeresPluginLibrary.json
ZeresPluginLibrary.config.json
ZeresPluginLibrary.raw
ZeresPluginLibrary.data.json
```

<Callout emoji="ℹ️">
    You might not have all these files, so just delete the ones you have.
</Callout>

3. Restart Discord again via the [system tray](https://www.tutorialspoint.com/windows10/windows10_system_tray.htm) icon.
4. Download [the plugin](https://raw.githubusercontent.com/rauenzi/BDPluginLibrary/master/release/0PluginLibrary.plugin.js). Open the hyperlink, and Press `Ctrl + S` to download it.
5. Copy and paste the downloaded file to your plugins folder, then restart Discord
again via the [system tray](https://www.tutorialspoint.com/windows10/windows10_system_tray.htm) icon.

That's all. Discord should no longer crash on launch!
